API Specification

GDG-X Hub API Version: v1, BaseURL: https://hub.gdgx.io/api/v1/
Status: alpha
  • All Routes support JSONP (just add your callback in the query, e.g. ?callback=myfunc)
  • Simple API Keys, OAuth & will be introduced later.
  • The API is rate limited. Please check the X-RateLimit-* headers for details
  • API Responses might be cached. Check the X-Cacher-Hit header.
  • Please file any bug reports and/or feature requests at https://github.com/gdg-x/hub/issues
